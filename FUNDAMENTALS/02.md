## Jupyter Notebook
주피터 노트북 단축키
- 마크다운 셀로 변환하기 : `esc` + `m` (명령모드로 변환 후 `m` )
- 코드 셀로 변환하기 : `esc` + `y` (명령모드로 변환 후 `y` )
- 셀의 실행 : `Shift` + `Enter`
- 셀 삭제 : `esc` + `x` or `esc` + `dd`
- 셀 삭제 취소 : `esc` + `z`
- 위에 셀 추가 : `esc` + `a`
- 아래에 셀 추가 : `esc` + `b`
